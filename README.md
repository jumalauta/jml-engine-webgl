# jml-engine-webgl
Jumalauta demo engine, WebGL edition

## Controls

| Key               | Action                    |
|-------------------|---------------------------|
| Enter             | Start demo                |
| Esc               | Stop demo                 |
| Alt + 1           | Rewind demo -1 second     |
| Alt + 2           | Rewind demo +1 second     |
| Alt + 3           | Pause demo                |
| Ctrl + Alt + Home | Rewind demo to start      |
| Ctrl + Alt + End  | Rewind demo to end        |
